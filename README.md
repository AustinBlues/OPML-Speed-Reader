# OPML_speed_reader

Fast OPML parser for importing RSS feeds.

## Getting started

	gem install opml_speed_reader

- [Examples](http://github.com/austinblues/opml_speed_reader/examples)

## Dependencies

- libxml-ruby - Gem
- libxml2 and libxml2-dev - GNOME library to manipulate XML files
